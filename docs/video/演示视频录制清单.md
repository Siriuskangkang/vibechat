# VibeChat 演示视频 · 录制清单（照着录）

> **怎么用**：按镜号顺序录，**每镜存一个独立文件**，照「操作步骤」逐动作点。录错**重录该镜即可**（分镜录的好处），不影响其他镜。
> **配套**：旁白/配音见 [`演示视频脚本.md`](./演示视频脚本.md)；本文件只管「录什么、怎么操作、怎么命名」。
> **核心高光**：镜 9–11 是你提的**分屏双端**（PC + 手机同输入→同房→聊天），是整条片的灵魂，多给时长。

---

## 0. 录制前准备（一次性，约 20 分钟）

- [ ] **线上烟测**：`https://vibechat.wukangkang.com` 跑通全链路一次。
- [ ] **PC 浏览器**：Chrome，登 `vibechat.wukangkang.com`，缩放 100%，关书签栏/通知，清一次 cookie（保证干净 session）。
- [ ] **手机**：iPhone/安卓，连 **USB** 到 Mac（分屏投屏用），手机浏览器打开同一地址。
- [ ] **双接口**：后端 `.env` 的 `LLM_PROVIDER` 能切 `openai`/`anthropic`，两路 key 配好（镜 13 用）。
- [ ] **录屏工具**：macOS 系统录屏 `⌘⇧5`（或 OBS）。分辨率 **1920×1080**，帧率 30。
- [ ] **投屏布局**（仅镜 9–11 用）：见 §分屏专项，提前摆好「左 Chrome / 右手机」。
- [ ] **桌面清理**：换干净壁纸，关掉一切通知/弹窗/狗头表情。

## 2. 分镜录制清单（逐动作）

### 镜 1 · 开场 · `01-开场.mp4`
- **设备**：PC 全屏
- **画面**：首页，星尘缓慢下落，标题渐显
- **操作**：① 打开首页 ② 手离开鼠标，静置 5s 让星尘动效展开
- **时长**：~8s（旁白开场）
- **注意**：不要动，纯氛围

### 镜 2 · 线上访问 · `02-线上访问.mp4`
- **设备**：PC
- **画面**：地址栏
- **操作**：① 新标签页 ② 地址栏输入 `vibechat.wukangkang.com` ③ 回车 ④ 等首页加载完，停留 2s
- **时长**：~10s
- **注意**：强调「公网可访问」——慢慢输入地址，让观众看清

### 镜 3 · 情绪输入 · `03-情绪输入.mp4`
- **设备**：PC
- **画面**：首页输入框
- **操作**：① 点击输入框 ② 输入 `明天要汇报，脑子停不下来，睡不着` ③ 停 1s 让观众看清 ④ 点击「分析」 ⑤ 等 loading 动效（~2s）
- **时长**：~12s
- **注意**：打字速度自然，别太快

### 镜 4 · AI 情绪识别 · `04-AI识别.mp4`
- **设备**：PC
- **画面**：结果卡（主情绪/向量/解读）
- **操作**：① 结果卡出现后停留 ② 鼠标缓慢移过各字段（焦虑 / valence / arousal / 解读）指给观众看 ③ 停 2s
- **时长**：~15s
- **注意**：数值以实际输出为准；鼠标移动要慢

### 镜 5 · 情绪可视化 · `05-情绪星图.mp4`
- **设备**：PC
- **画面**：`EmotionColorWheel` 涟漪落点
- **操作**：① 让落点的涟漪+光晕动效跑一圈 ② 停留 2s
- **时长**：~12s

### 镜 6 · 自动匹配 · `06-自动匹配.mp4` ★重点

### 镜 9 · 分屏双端进入 · `09-分屏双端进入.mp4` ★★★核心
- **设备**：PC + 手机（分屏，先按 §分屏专项 摆好布局）
- **叙事**：换场景——「两个不同设备的人，同样的心情，能否相遇？」
- **操作**：
  1. **左 Chrome**：回首页（新标签或刷新）
  2. **右 手机**：浏览器打开 `vibechat.wukangkang.com`
  3. **左 Chrome**：输入 `加班到现在，脑子完全转不动了` → 点分析 → 等匹配 → 进「疲惫放空」房
  4. **右 手机**：输入**相同的话** `加班到现在，脑子完全转不动了` → 分析 → 进**同一间房**
  5. 此时两端都在「疲惫放空」房，画面同时显示两个匿名身份 → **停留 3s**，让观众看清「两人同处一室」
- **时长**：~25s
- **注意**：两端输入必须**逐字相同**（这是「同频→同房」的证明）；操作可先后，不必同步

### 镜 10 · 分屏跨端聊天 · `10-分屏跨端聊天.mp4` ★★★核心
- **设备**：PC + 手机（分屏，接镜 9 状态）
- **操作**：
  1. **左 Chrome**：输入 `还在赶吗` → 发送
  2. **右 手机**：实时收到该消息（气泡浮入）→ 停 1s
  3. **右 手机**：输入 `嗯，一起熬过去` → 发送
  4. **左 Chrome**：实时收到 → 停 1s
  5. 再来一轮：左发 `辛苦了，共勉` → 右收到；右发 🌙 表情或 `晚安` → 左收到
- **时长**：~25s
- **注意**：这是规避「聊天无法收发」扣分项的关键镜；每条发出后**等对端收到再发下一条**，节奏清楚

## 3. 分屏专项（镜 9–11 必读）

**目标**：一次录屏得到「左 Chrome / 右手机」分屏画面，后期不用合成。

**布局方法（macOS + iPhone，推荐）**：
1. iPhone 用 **USB 连 Mac**。
2. 打开 **QuickTime Player** → 菜单「文件 → 新建影片录制」→ 录制按钮旁下拉 → **摄像头选你的 iPhone**（此时 QuickTime 窗口实时显示手机画面，**先别点录制**，只当投屏显示器用）。
3. 把 QuickTime 手机窗口拖到屏幕**右半边**，Chrome 浏览器拖到**左半边**，各占一半。
4. 用 **macOS 系统录屏 `⌘⇧5`**（或 OBS）录**全屏** → 一次得到分屏画面。

**安卓**：用 `scrcpy` 投屏到 Mac 窗口，布局同上。

**操作时序**：分屏镜里两端要轮流操作——录屏开着，先在 Mac 上点左边，再拿手机点右边；两端画面始终同框可见，观众一眼看出是两个真设备在实时通信。

> 如果投屏麻烦，**降级方案**：镜 9–11 改录「左 Chrome / 右无痕窗口」（两个浏览器窗口并排），叙事不变，只是「真设备感」弱一点。但分屏真机效果明显更好，建议优先试投屏。

## 4. 录制技巧

- **静帧余量**：每镜开头和结尾各留 1–2s 不动，后期裁切和拼接才稳。
- **出错重录**：分镜录的好处——某镜错了只重录那一个文件，覆盖同名即可。
- **节奏**：重点镜（⑥匹配 / ⑨⑩分屏 / ⑪情绪场 / ⑬双接口）操作慢、停留足；过渡镜可快。
- **真实数据**：旁白里的「87%」「焦虑」等以页面实际输出为准，录制时记下真实数值，配音阶段同步。
- **不说话**：全程静音录画面，配音后期 TTS 加，避免环境噪音和口误。
- **总时长**：16 镜合计约 4:30，后期微调落在 3–6 分钟内。
