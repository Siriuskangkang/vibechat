```markdown
# VibeChat — AI 驱动的情绪社交

输入此刻心情，AI 解析情绪色彩，把你匹配进一间情绪同频的匿名房间；AI 主持破冰、引导对话，离场给出情绪旅程。让"当下情绪"决定你和谁相遇。

🌐 **在线体验：https://vibechat.wukangkang.com**

## 功能
- 情绪输入 + AI 结构化分析（标签/向量/强度/正负向/解读）
- VAD 向量匹配进最近情绪房间 + 同频度 + 情绪色盘可视化
- WebSocket 匿名多人群聊（AI 主持破冰 + 兜底 bot）
- 房间集体情绪场（色场随成员情绪流动）+ 共鸣度
- 离场情绪旅程（start→mood_end 轨迹）
- 极端情绪求助卡片

## 技术栈
- 前端：Next.js (App Router) + React + TypeScript + Tailwind，纯 SVG/CSS 可视化
- 后端：FastAPI (async) + SQLAlchemy 2.0 + SQLite
- LLM 双标准适配：OpenAI 标准（百炼/Qwen）+ Anthropic 标准（DeepSeek `/anthropic`），环境变量切换

## 双接口配置（★ 重点）

通过 `backend/.env` 的 `LLM_PROVIDER` 一键切换两种标准接口：

\`\`\`bash
LLM_PROVIDER=openai            # openai | anthropic

# OpenAI 标准（百炼）
OPENAI_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
OPENAI_API_KEY=sk-***
OPENAI_MODEL=qwen-plus

# Anthropic 标准（DeepSeek 兼容端点）
ANTHROPIC_BASE_URL=https://api.deepseek.com/anthropic
ANTHROPIC_API_KEY=sk-***
ANTHROPIC_MODEL=deepseek-chat
\`\`\`

切换后无需改任何业务代码。日志会打印当前 `provider`/`model`。
```

- [ ] **Step 5: nginx server block（复用现有 SSL 证书路径）**

`/etc/nginx/sites-available/vibechat.conf`（证书路径按 Step 1 实际填写）：
```nginx
server {
    listen 443 ssl http2;
    server_name vibechat.wukangkang.com;
    ssl_certificate     /path/to/existing.crt;
    ssl_certificate_key /path/to/existing.key;

    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
    location /api/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
    }
    location /api/ws/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_read_timeout 86400;
    }
}
```

```bash
ssh kang '
ln -s /etc/nginx/sites-available/vibechat.conf /etc/nginx/sites-enabled/
nginx -t && systemctl reload nginx
systemctl enable --now vibechat-backend vibechat-frontend
'
```
